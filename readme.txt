fastwidget отображает на странице фиксированное меню.
Для его подключения необходимо в html-файл необходимой страницы перед закрывающим тегом body поместить следующую строчку:
<script defer src="fastwidget.js"></script>
При этом файл виджета fastwidget.js должен находиться в одном каталоге с файлом страницы. Если поместить его в другое место, нужно соответственно скорректировать адрес в строке запуска.
